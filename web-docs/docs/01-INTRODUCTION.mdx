## Description

A library of extensions to the functionality of the Ewon Flexy Java environment (Ewon ETK), created
by the HMS Networks, MU Americas Solution Center.

## Getting Started

To get started, please refer to the [Quick Start Guide](03-QUICK_START_GUIDE.mdx). This guide
provides instructions for using the library, and how to access it via the Solution Center's
Maven repository.
